Reset the Demo 2 workshop components (nonCompliantCard and nonCompliantAlert) to their original non-compliant state.

This restores the components from the backup directory so participants can redo the SLDS 2 transition exercise.

```bash
npm run reset-demo2
```

After reset:
- The non-compliant components will have all their original SLDS violations restored
- Refresh your browser to see the changes
- Components are ready for the uplift exercise again
